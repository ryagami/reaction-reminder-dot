# Reaction Reminder Dot

Tiny Foundry VTT module for dnd5e on v13 that adds a clickable reaction indicator dot to each combatant in the Combat Tracker.

## Features

- Manual reaction toggle per combatant in the tracker
- Automatic reset to available at each combatant's turn start
- Optional auto-marking when a dnd5e reaction activity is used
- Player toggling can be restricted to GM only

## Install (local development)

1. Copy this folder into your Foundry Data `modules` directory.
2. Enable **Reaction Reminder Dot** in your world.
3. Open Module Settings to configure auto-tracking and player permissions.

## Notes

- Built for Foundry VTT v13.
- Intended for dnd5e worlds.
